# Combining Estimators — Statistics for Data Science Project
**University of Pisa | A.Y. 2023/24**

## Overview
This project was developed for the Statistics for Data Science course 
at the University of Pisa. It is based on the paper:

> *A general procedure to combine estimators* — F. Lavancier, P. Rochet 
> (Computational Statistics & Data Analysis, 2015)

## Description
The paper proposes a general method to combine multiple estimators of 
the same parameter using a weighted average, where optimal weights 
minimize the quadratic loss (MSE). We studied, presented, and 
partially re-implemented the method in R.

## Topics covered
- Estimator averaging and oracle optimality
- MSE matrix estimation (parametric bootstrap, non-parametric bootstrap)
- Applications: symmetric distribution, Weibull model, Boolean model, quantile estimation

## Repository contents
- `presentation.pdf` — project slides (in English)
- `script.R` — R script with re-implementation of methods and experiments

## Technologies
- R
- Statistical methods: MLE, Method of Moments, OLS, Bootstrap
